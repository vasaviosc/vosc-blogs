# to add a blog

- create your \[slug\].md file in the format

```md
---
title: "title goes here"
date: 2026-02-14 08:21
author: "bhushanwho"
---

# waow blog

such blog! waow!
```

- find the template at `template.md`

- fork the repo and create a PR with your \[slug\].md in the `blogs` folder.

- your blog (will be reviewed, merged and) pushed into the repo automatically thanks to `.github/workflows/push-blogs.yml`
